About the Webserver Project

This is a custom C++ webserver built from scratch as part of the Hive Helsinki curriculum. The goal is to understand how HTTP works by implementing request parsing, response handling, routing, and static file serving without external frameworks.

Key features include:

  • Support for multiple clients with non-blocking I/O
  • Serving static HTML, CSS, and images
  • Custom request parser and response builder
  • Error handling with status codes (200, 404, etc.)
  • POST request handling with form submissions
  • File upload system (1MB limit)
  • File deletion system
  • Multipart form-data processing
