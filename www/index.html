About the Webserver Project

This project is a custom C++ webserver built from scratch as part of the Hive Helsinki curriculum. The goal is to understand how HTTP works under the hood by implementing request parsing, response handling, routing, and static file serving without relying on external frameworks.

Key features include:

  • Support for multiple clients using non-blocking I/O
  • Serving static HTML, CSS, and images
  • Custom request parser and response builder
  • Error handling and status codes (200, 404, etc.)
  • POST request handling with form submissions
  • File upload system with 1MB size limit
  • Multipart form-data processing
