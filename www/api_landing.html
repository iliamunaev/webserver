API Server Landing Page

Welcome to the API server. This server handles REST API endpoints, file uploads, and administrative functions.

Available Endpoints

  • GET /api - API response endpoint (api_response.html)
  • GET/POST /uploads - File upload and management
  • GET/POST /admin - Administrative functions (admin_dashboard.html)
  • GET /errors - Error pages directory
  • GET /api/v1 - Redirects to /api

File Operations

  • POST /uploads - Upload files (max 1MB)
  • DELETE /uploads/filename - Delete uploaded files
